Archivos de código fuente adicionales

Para compilar una versión con todas las características de TestDisk y Photorec, podrías necesitar los siguientes archivos:

  • e2fsprogs-1.39.tar.gz (Debería funcionar con versiones anteriores)
  • progsreiserfs-0.3.1-rc8.tar.gz
  • ntfsprogs-1.13.1.tar.gz (TestDisk 5.5 ó posterior no funciona con versiones más viejas que 1.9.4)
