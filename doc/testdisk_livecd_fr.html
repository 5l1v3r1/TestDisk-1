TestDisk && Live rescue cd

TestDisk (pas toujours la dernière version) est inclu dans :

  • GParted LiveCD
  • Knoppix (Astuce sur l'utilisation de TestDisk sous Knoppix)
  • Parted Magic
  • PLD RescueCD
  • Recovery Is Possible Linux rescue system
  • Slax-LFI Version 6.6 sur Slax-LFI 1.2
  • System Rescue CD
  • Ultimate Boot CD

Notez que TestDisk est également disponible dans les distributions suivantes

  • ALT Linux backports
  • Debian contrib
  • Fedora Extras
  • FreeBSD ports
  • Gentoo
  • Gentoo Portage
  • Mandriva contrib (ex-Mandrake)
  • PLD Linux distribution
  • Source Mage GNU/Linux
  • Ubuntu

Certaines d'entre elles ont un mode de sauvetage qui peut être utilisé avec TestDisk.
