TestDisk && Moteurs de Recherche

Pour aider TestDisk à avoir un meilleur classement dans les moteurs de recherche, créer un lien sur votre site web vers l'URL de TestDisk.


TestDisk && Documentation

Votre aide est la bienvenue. Contribuez à la documentation de TestDisk.

La Mailing List TestDiskDevelopment

Si vous désirez vous abonner à la mailing list TestDiskDev , envoyez un mail à sympa@sympa.global-asp.net avec subscribe testdisk-dev dans le corps ou l'objet du message. Notre mailing list inclut:

  • les découvertes de bugs et le développement de TestDisk
  • le développement de nouvelles fonctionnalités
  • des articles sur de nombreux autres outils de récupération de données

Nous recherchons des personnes pour intégrer l'équipe de développement qui sont:

  • familiers de la programmation
  • familiers de la récupération de données ou
  • intéressés pour faire des tests sur des ordinateurs dont ils ont l'accès
