Instalación de libjpeg usando Fink

Puede pasar por el proceso de descargar, configurar, y compilar esto por usted mismo, pero, si está usando Fink, no hace realmente falta. Fink automatiza la instalación de docenas de paquetes UNIX y es extremadamente fácil de utilizar.

Siga estas instrucciones y terminará con un nuevo directorio llamado /sw con un montón de software en él. Ahroa puede usar Fink para instalar bibliotecas con un solo comando. A continuación se explica como se instala libjpeg: 

% sudo /sw/bin/fink install libjpeg
