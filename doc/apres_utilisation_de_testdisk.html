Ecriture de la table de partition

Mac OS X

Utiliser pdisk pour recréer la carte des partitions Mac, utiliser les valeurs données par TestDisk

Comment vérifier et réparer le système de fichiers

ReiserFS

Si le superblock d'une partition reiserfs est manquant, il peut être reconstruit avec reiserfsck --rebuild-sb device.

Pour réparer un système de fichiers FAT/NTFS sous Windows

Pour vérifier une partition NTFS sous Windows, lancez chkdsk /f c:


Comment rendre le système bootable

Dos - Win9x

Si votre système d'exploitation ne démarre pas, vous pouvez réinstaller le système de fichiers avec la commande: sys c:

Windows 2000/XP/2003

Si votre Windows ne démarre pas, essayer fixmbr de la console Windows et fixboot pour réparer le secteur de boot NTFS.

Linux/FreeBSD

Vous devez mettre à jour votre /etc/fstab pour vérifier le nouvel ordre des partitions.

Multiboot

Vous devez mettre à jour la configuration de votre multiboot et le réinstaller dans le Master Boot Record(Disque principal d'initialisation).

  • Lilo: /etc/lilo.conf, lilo pour réinstaller
  • Grub: /boot/grub/grub.conf, grub-install device
  • Windows (NT/2000/XP/2003/...): c:\boot.ini, lancez fixmbr de la console de récupération(Recovery Console) (ie. fixmbr \Device\HardDisk0)

Comment récupérer des fichiers

Ext2fs

Pour récupérer des fichiers sous linux, vous pouvez essayer e2undel

PhotoRec

Essayez PhotoRec si vous voulez récupérer des fichiers et si le système est trop endommagé pour être réparé.
