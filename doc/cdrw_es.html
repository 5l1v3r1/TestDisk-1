Diseño de un CD

Un CD está compuesto de varias zonas

  • PCA (Power Calibration Area)
  • PMA (Program Memory Area)
  • una ó más sesiones

La SUA (System User Area) está compuesta de PCA y PMA.

Cada sesión consiste de

  • un area de Lead-In, conteniendo la Table of Contents (TOC) de la sesión,
  • la Program Area en la cual las pistas individuales están ubicadas,
  • y el area de Lead-Out.

Una pista está compuesta de

  • una zona de pre-gap
  • un area de audio/user data
  • una zona de relleno (padding)

Cada bloque ó sector tiene una cabecera, la ubicación del bloque está guardada en cada cabecera.

Recuperación de CD-R/CR-RW/DVD rallados

Photorec funciona bien pero los sectores dañados hacen que el proceso de recuperación de datos sea mucho más lento.

Recuperación de CDs blanqueados rápidamente

¡Es posible recuperar datos de un CD borrado en modalidad rápida (quick-erase) sin modificarlo!

Cuando un CD-RW es borrado en esta modalidad, el PMA, TOC, pregap y los primeros sectores son eliminados. Ya que la TOC ha sido borrada, el CD-RW aparece como vacío. Ya que los primeros sectores han sido borrados, incluyendo las cabeceras, los sectores 0, 1, 2 ... no pueden ser ubicados pero sectores subsequentes sí pueden ser encontrados. Desafortunadamente, no todos los sistemas operativos permiten el acceso a dichos sectores, pero Linux sí.

Para recuperar los datos perdidos, ejecute la versión de Photorec para Linux. Si el tamaño listado del CD-RW es 0 ó 2048 bytes, el firmware del CDROM va a bloquear las siguientes lecturas, así que deberá utilizar otra lectora de CDROM. El comienzo de la recuperación es bastante lento porque los primeros sectores son ilegibles pero generalmente después del sector 300 los datos se pueden recuperar, así que sea paciente.

Lectura de sesiones previas

Con un CD multi sesión, es posible borrar archivos de sesiones previas. Debido a que los archivos no son realmente eliminados, es posible recuperarlos. Para leer archivos de una primera sesión ejecute bajo Linux 

mount /dev/cdrom /mnt/cdrom -t iso9660  -o session=0
