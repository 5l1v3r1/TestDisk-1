Fedora, RedHat Enterprise, CentOS

Under a Linux RedHat, you need to install

  • ncurses, ncurses-devel
  • e2fsprogs, e2fsprogs-devel
  • libjpeg, libjpeg-devel
  • ntfsprogs ntfsprogs-devel 
yum install ncurses ncurses-devel e2fsprogs e2fsprogs-devel libjpeg libjpeg-devel ntfsprogs ntfsprogs-devel

To use all TestDisk functionnalities, you have to compile Reiserfs libraries.

Linux distribution using RPM

To install TestDisk rpm, run 

rpm -Uvh testdisk-*.i386.rpm

If it doesn't work, you may have to rebuild the RPM for your distribution using 

rpmbuild --recompile testdisk-5.2-1.src.rpm

Debian

Under Debian, install

  • libncurses5, libncurses5-dev
  • libntfs5, libntfs-dev
  • e2fslibs, e2fslibs-dev
  • libreiserfs0.3

They can be installed via apt-get: 

apt-get install e2fslibs-dev libncurses5-dev libntfs-dev libjpeg62-dev libreiserfs0.3-dev
