Norton GoBack (anteriormente conocido como Wildfile GoBack, Adaptec GoBack, Roxio GoBack) es una utilidad de disco basada en Microsoft Windows. Si ejecuta la versión de Windows de TestDisk, necesita deshabilitar o desinstalar antes GoBack para poder escribir en la tabla de partición.
