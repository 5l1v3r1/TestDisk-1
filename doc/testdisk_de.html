TestDisk

TestDisk ist OpenSource-Software und lizensiert unter der GNU Public License (GPL).

TestDisk ist ein leistungsstarkes freies Daten-Wiederherstellungs-Utility! Es wurde in erster Linie erstellt um verlorene Partitionen wiederherzustellen und/oder nicht bootende Festplatten wieder bootfähig zu machen, wenn das Problem durch fehlerhafte Software, bestimmte Arten von Viren oder menschliche Fehler (wie versehentliches Löschen der Partitionstabelle) verursacht wird. Partitionstabellen wiederherstellen mit TestDisk ist wirklich einfach.

Betriebssysteme

TestDisk läuft unter

  • DOS (entweder im realen Dos-Modus oder in einer Windows 9x-DOS-Box),
  • Windows (NT4, 2000, XP, 2003),
  • Linux,
  • FreeBSD, NetBSD, OpenBSD,
  • SunOS und
  • MacOS

Quelldateien und vorkompilierte ausführbare Dateien sind verfügbar für DOS, Win32, MacOSX und Linux auf der Herunterladen Seite

Dateisysteme

Testdisk findet verlorene Partitionen für folgende Dateisysteme:

  • BeFS ( BeOS )
  • BSD disklabel ( FreeBSD/OpenBSD/NetBSD )
  • CramFS, Compressed File System
  • DOS/Windows FAT12, FAT16 und FAT32
  • HFS and HFS+, Hierarchical File System
  • JFS, IBM's Journaled File System
  • Linux Ext2 und Ext3
  • Linux Raid
    • RAID 1: mirroring
    • RAID 4: striped array with parity device
    • RAID 5: striped array with distributed parity information
    • RAID 6: striped array with distributed dual redundancy information
  • Linux Swap (versions 1 und 2)
  • LVM and LVM2, Linux Logical Volume Manager
  • Mac partition map
  • Novell Storage Services NSS
  • NTFS ( Windows NT/2K/XP/2003 )
  • ReiserFS 3.5, 3.6 und 4
  • Sun Solaris i386 disklabel
  • Unix File System UFS und UFS2 (Sun/BSD/...)
  • XFS, SGI's Journaled File System

Zusammenfassung

TestDisk fragt das Bios oder das Betriebssystem ab um Festplatten und ihre Charakteristiken zu finden ( LBA-Größe und CHS-Geometrie). TestDisk führt dabei eine schnelle Überprüfung der Festplattenstruktur aus und vergleicht diese mit der Partitionstabelle auf Eintragsfehler. Wenn die Partitionstabelle Eintragsfehler hat, kann TestDisk diese reparieren. Bei fehlenden Partitionen oder einer völlig leeren Partitionstabelle, kann TestDisk nach Partitionen suchen und wenn notwendig eine neue Tabelle oder sogar einen neuen MBR erzeugen.

Wie auch immer, kann der Anwender die Liste der von TestDisk gefundenen und für eine Wiederherstellung möglichen Partition(en) anschauen und die Partition(en) auswählen, welche vorher funktioniert hatte(n), bevor die Festplatte nicht mehr startete oder die Partition(en) verloren ging(en). In einigen Fällen, speziell nach einer ausführlichen Suche nach verlorenen Partitionen, kann TestDisk Partitionsdaten anzeigen die Überbleibsel von einer Partition sind, die vor langer Zeit gelöscht und überschrieben wurde.

TestDisk hat Charakteristiken für Anfänger und Fortgeschrittene. Für Anwender die nur wenig oder nichts über Datenwiederherstellungtechniken wissen, kann TestDisk eingesetzt werden, um ausführliche Informationen über eine nicht startende Festplatte zu sammeln, um diese an einen Techniker für weitere Analyse zu schicken. Diejenigen die sich mit diesen Prozeduren besser auskennen, werden in TestDisk ein praktisches Tool für eine Vor-Ort-Wiederherstellung finden.
