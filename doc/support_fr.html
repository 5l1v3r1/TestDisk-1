Support technique de TestDisk

Si vous avez besoin d'aide concernant l'utilisation de TestDisk,

  • exécuter testdisk_win.exe ou testdisk,
  • choisisser 'Create a new log file',
  • choisisser votre disque dur puis le type de partionnement,
  • puis sélectionner Analyse and Search!,
  • pour chaque partition, utiliser 'p' pour lister les fichiers,
  • à la fin, sélectionner Quit,

Envoyez un email à grenier@cgsecurity.org avec

  • le fichier testdisk.log,
  • une brève explication de votre problème et
  • quelques informations (taille, nom de volume, système de fichier) sur vos précédentes partitions.

Les extensions de fichiers pour les types de fichiers connus sont masquées par défaut par l'explorateur de Windows, modifier ce paramétrage si vous avez des difficultés pour trouver le fichier testdisk.log.

Support technique de PhotoRec

Si PhotoRec ne retrouve pas certain type de fichier, envoyez à grenier@cgsecurity.org

  • un extrait du fichier qui n'est pas encore reconnu ou
  • quelques images non modifiées prises avec votre appareil photo numérique ainsi que le modèle exact.
