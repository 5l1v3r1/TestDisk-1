Escribiendo la tabla de particiones

Mac OS X

Use pdisk para recrear el mapa de partición de Mac usando los valores dados por TestDisk

Como revisar y reparar un sistema de archivos

ReiserFS

Si el superblock de una partición reiserfs está perdido, puede ser reconstruído reiserfsck --rebuild-sb device.

Para reparar un sistema de archivos FAT/NTFS bajo Windows

Para revisar una partición NTFS desde Windows, ejecute chkdsk /f c:


Como hacer el sistema Booteable nuevamente

Dos - Win9x

Si su sistema no inicia, puede reinstalar los archivos de sistema con sys c:.

Windows 2000/XP/2003

Si su Windows no inicia, intente fixmbr desde la Recovery Console de Windows y fixboot para reparar el sector de booteo de NTFS.

Linux/FreeBSD

Puede tener que actualizar su /etc/fstab para reflejar el nuevo orden de partición.

Multiboot

Puede tener que actualizar su configuración de multiboot y reinstalar el multiboot en el Master Boot Record (MBR).

  • Lilo: /etc/lilo.conf, lilo para re instalar
  • Grub: /boot/grub/grub.conf, dispositivo grub-install
  • Windows (NT/2000/XP/2003/...): c:\boot.ini, ejecute fixmbr desde la Recovery Console (ejemplo. fixmbr \Device\HardDisk0)

Como recuperar (undelete) archivos

Ext2fs

Para recuperar archivos en linux, puede probar e2undel

ReiserFS

Lea el ReiserFS file undelete HOWTO

PhotoRec

Intente PhotoRec si quiere recuperar archivos y el sistema está demasiado dañado para ser reparado.
