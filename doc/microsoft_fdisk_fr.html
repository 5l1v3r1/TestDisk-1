Microsoft Fdisk

Quand il est utilisé, Microsoft FDISK détruit quelques données sur votre disque. Pendant que le message checking disk integrity est affiché, FDISK écrit le caractère hexadécimal F6 sur les premier et septième secteurs de chaque piste. Si vous avez accidentellement effacé un volume, ne lancez aucun programme de Microsoft FDISK!
