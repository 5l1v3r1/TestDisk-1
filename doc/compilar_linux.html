RedHat

En Linux RedHat, necesita instalar

  • ncurses, ncurses-devel
  • e2fsprogs, e2fsprogs-devel 
rpm -Uvh ncurses-*.rpm ncurses-devel-*.rpm e2fsprogs-*.rpm e2fsprogs-devel-*.rpm libjpeg-*.rpm libjpeg-devel-*.rpm

Para usar todas las funcionalidades de TestDisk, tiene que compilar las bibliotecas de Reiserfs y NTFS.

Distribución de Linux usando RPM

Para instalar el rpm de TestDisk, ejecute 

rpm -Uvh testdisk-*.i386.rpm

Si no funciona, puede ser que tenga que reconstruir el RPM para su distribución usando 

rpmbuild --recompile testdisk-5.2-1.src.rpm

Debian

En Debian, instalar

  • libncurses5, libncurses5-dev
  • libntfs5, libntfs-dev
  • e2fslibs, e2fslibs-dev
  • libreiserfs0.3

Pueden ser instaladas vía apt-get.
