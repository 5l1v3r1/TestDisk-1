Delete

Supprime la liste des partitions dans la Table des Partitions Intel (en mettant des zéros à la place). Le code exécutable du MBR Master Boot Record et les octets de signature restent les mêmes.

Certains BIOS utilisent le contenu de la table de partition pour définir la géométrie du disque dur. Certains valeurs inattendues peuvent déclencher des bugs du BIOS. La version DOS de TestDisk est la plus affectée. Si elle détecte un bug connu, elle affiche Buggy BIOS après la géométrie du disque. Delete peut résoudre le problème. Redémarrez votre ordinateur et faites détecter votre disque dur.
