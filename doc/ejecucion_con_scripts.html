TestDisk y PhotoRec pueden ejecutarse automáticamente usando sus propios comandos built-in. Un archivo de script (como un .cmd ó .bat 'Batch files' bajo MS-DOS/Windows, ó algún archivo de shell en Linux) también puede ser de ayuda.

Este documento describe la sintaxis de la línea de comandos.

TestDisk

Sintaxis: testdisk /cmd device cmd

Selección de Dispositivo

Usar el nombre del dispositivo, por ejemplo /dev/hda, /dev/hdb, /dev/sda.
Para la versión DOS, usar /dev/sda128 para el primer disco, /dev/sda129 para el segundo y así...

Selección del tipo de Partición

  • partition_i386
  • partition_mac
  • partition_none
  • partition_sun
  • partition_xbox

Menú Principal

  • advanced
  • analyze
  • delete
  • geometry
  • mbr_code
  • options
  • list

Menú de Análisis

  • backup
  • mode_vista: buscar particiones creadas por el Administrador de Windows Vista
  • list
  • search
  • noconfirm
  • write

Menú Avanzado

  • type
  • boot
  • superblock
  • number: el número de la partición a elegir

Menú de FAT1x

  • rebuildbs
  • dump
  • repairfat
  • initroot

Menú de FAT32

  • rebuildbs
  • dump
  • repairfat
  • originalfat
  • backupfat

Menú de reconstrucción de FAT

  • list
  • dump

Menú de NTFS

  • rebuildbs
  • dump
  • originalntfs
  • backupntfs
  • repairmft

Menú de reconstrucción de NTFS

  • list
  • dump

Menú de Geometría

  • C,number of cylinders
  • H,number of heads
  • S,number of sectors
  • N,sector size

Algunos Ejemplos 

testdisk /debug /log /cmd /dev/hda analyze,search

testdisk /debug /log /cmd partition.dd options,partition_none,geometry,H,32,analyze,list,advanced,boot,rebuildbs,
list partition.dd

PhotoRec

Sintaxis: photorec /cmd device cmd

Selección de tipo de Partición

  • partition_i386
  • partition_mac
  • partition_none
  • partition_sun
  • partition_xbox

Menú Principal

  • number: the partition number to select
  • search
  • options

Menú de Opciones

  • mode_ext2

Algunos Ejemplos 

photorec /debug /log /cmd /dev/hdb select,search
photorec /debug /log /cmd /dev/hda options,mode_ext2,5,search
