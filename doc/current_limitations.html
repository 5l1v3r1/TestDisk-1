Limitations

TestDisk and PhotoRec are using a 64 bits internal data offset representation.
IDE LBA28 adressing is limited to 128GB HD.
IDE LBA48 adressing is limited to 131072 TB.
PC/Intel MBR partition table is limited to 2TB.
GUID Partition Table (GPT) can be up to 18 exabytes (Not yet supported by TestDisk).
Most common limitation is to have an OS that doesn't handle IDE LBA48 adressing mode.
