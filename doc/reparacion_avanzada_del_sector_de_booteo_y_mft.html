Reparación del Sector de Booteo NTFS

Si el sector de booteo del NTFS está dañado, los datos no se pueden acceder. Windows preguntará La unidad no tiene formato,¿Desea darle formato ahora? El comando mount de Linux mostrará wrong fs type, bad option, bad superblock

TestDisk le permite manipular el sector de booteo de las particiones NTFS. Seleccione la partició que quiera modificar y elija Boot.

Recuperando el sector de booteo NTFS en particiones NTFS usando su copia de backup

TestDisk puede usar el sector de booteo de backup para arreglar el sector de booteo NTFS. El sector de booteo primario es el sector cero del sistema de archivos, el sector de backup de booteo está ubicado cerca del final del sistema de archivos. TestDisk comparar el sector de booteo con el sector de booteo de backup. Si no concuerdan, puede sobreescribir el sector de booteo NTFS con el de backup (Backup BS) ó viceversa (Org. BS). Dump puede ser usado para mostrar el contenido del sector de booteo en hexadecimal ó ASCII.

Reconstruyendo el sector de booteo NTFS en particiones NTFS

Si el sector de booteo NTFS y el sector de backup de booteo están dañados, es necesario reconstruir el sector de booteo NTFS. TestDisk busca el MFT (Master File Table: $MFT) y su backup ($MFTMirr). Lee el tamaño del registro MFT y computa el tamaño del cluster, lee el tamaño del Index Allocation Entry en el directorio raíz indice, puede proveer un nuevo sector de booteo. TestDisk permite al usuario listar los archivos antes de escribir.

Repair NTFS MFT

The MFT (Master File Table) is sometimes corrupted. If Microsoft Check Disk (chkdsk) failed to repair the MFT, run TestDisk and in the Advanced menu, select your NTFS partition and choose Repair MFT. TestDisk will try to repair the MFT using MFT mirror, its backup.
